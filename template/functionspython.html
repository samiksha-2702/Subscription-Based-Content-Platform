Functions & OOP Concepts

Functions

  • Define a function using def
  • Functions help reuse code and improve readability
  • Return values using return
  • Use parameters to pass data into functions
  • Functions can return multiple values
  • Nested functions are also supported in Python

def greet(name):
    return f"Hello {name}"

print(greet("Alice"))

Important Interview Point

Python supports first-class functions, meaning functions can be assigned to variables, passed as arguments, and returned from other functions.

Types of Function Arguments

  • Positional Arguments
  • Default Arguments
  • Keyword Arguments
  • Variable Length Arguments
  • *args → multiple positional arguments
  • **kwargs → multiple keyword arguments
  • Optional parameters supported
  • Argument unpacking supported

def student(name, age=20):
    print(name, age)

student("Harshali")


def total(*nums):
    return sum(nums)

print(total(10,20,30))

Lambda Functions

  • Anonymous single-line functions
  • Useful for short operations
  • Mostly used with map(), filter(), and sorted()
  • Improves concise coding style

square = lambda x: x * x

print(square(5))


nums = [1,2,3,4]

result = list(map(lambda x: x*2, nums))

print(result)

Interview Question

Difference between normal functions and lambda functions is commonly asked in interviews.

Object Oriented Programming (OOP)

  • Class → Blueprint for objects
  • Object → Instance of class
  • Encapsulation → Binding data & methods together
  • Inheritance → Reusing parent class properties
  • Polymorphism → Same method behaves differently
  • Abstraction → Hiding internal implementation

class Person:

    def __init__(self, name):
        self.name = name

    def greet(self):
        print(f"Hello {self.name}")

p = Person("Alice")
p.greet()

Constructors & Self Keyword

  • __init__() is constructor method
  • Automatically called when object is created
  • self refers to current object instance
  • Used to access class variables and methods

class Car:

    def __init__(self, brand):
        self.brand = brand

c = Car("BMW")

print(c.brand)

Inheritance

  • Single Inheritance
  • Multiple Inheritance
  • Multilevel Inheritance
  • Hierarchical Inheritance

class Person:

    def __init__(self, name):
        self.name = name

class Student(Person):

    def __init__(self, name, roll):
        super().__init__(name)
        self.roll = roll

s = Student("Rahul",101)

print(s.name)
print(s.roll)

Polymorphism

  • Method Overriding supported in Python
  • Operator overloading available
  • Same method name behaves differently

class Bird:

    def sound(self):
        print("Bird Sound")

class Dog(Bird):

    def sound(self):
        print("Bark")

d = Dog()
d.sound()
