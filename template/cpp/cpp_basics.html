C++ Fundamentals

What is C++?

  • C++ is a powerful general-purpose language
  • Developed by Bjarne Stroustrup
  • Supports procedural and object-oriented programming
  • Used in game engines, OS, and software development
  • Fast and performance-oriented language

Basic Program Structure

#include <iostream>
using namespace std;

int main(){

    cout << "Hello World";

    return 0;
}
  • iostream → Input Output library
  • main() → Starting point
  • cout → Output statement
  • return 0 → Successful execution

Variables & Data Types

  • int → Integer values
  • float → Decimal values
  • double → High precision decimal
  • char → Single character
  • bool → True or False
  • string → Text values
int age = 20;
float marks = 88.5;
char grade = 'A';

Input & Output

Using cin and cout

int x;

cin >> x;

cout << x;
  • cin → Accepts input
  • cout → Displays output
  • Uses stream operators

Multiple Inputs

int a, b;

cin >> a >> b;

cout << a + b;
  • Accept multiple values together
  • Useful in competitive programming

Escape Sequences

  • \n → New line
  • \t → Tab space
  • \" → Double quote
  • \\ → Backslash

Operators & Logic

Arithmetic Operators

  • Addition → +
  • Subtraction → -
  • Multiplication → *
  • Division → /
  • Modulus → %

Relational Operators

  • == Equal to
  • != Not equal to
  • > Greater than
  • < Less than
  • >= Greater or equal
  • <= Less or equal

Logical Operators

  • && → AND
  • || → OR
  • ! → NOT
  • Used in conditions and loops

Control Statements

If Else

int age = 18;

if(age >= 18){

    cout << "Adult";

}
else{

    cout << "Minor";

}

For Loop

for(int i=1; i<=5; i++){

    cout << i;

}
  • Repeats code multiple times
  • Useful for iteration

While Loop

int i = 1;

while(i <= 5){

    cout << i;

    i++;
}
