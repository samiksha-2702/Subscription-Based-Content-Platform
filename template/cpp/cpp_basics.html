Overview

What is C++?

C++ is a powerful general-purpose programming language created by Bjarne Stroustrup. It supports procedural, object-oriented, and generic programming. It is widely used in system development, game engines, competitive programming, operating systems, and high-performance applications.

Fundamentals

Basic Structure of a C++ Program

#include <iostream>
using namespace std;

int main() {
    cout << "Hello World";
    return 0;
}
  • include iostream → standard input/output library
  • using namespace std → avoids writing std:: repeatedly
  • main() → program execution starts here
  • cout → output statement
  • return 0 → successful execution
Core Concepts

Variables and Data Types

C++ provides multiple data types to store different kinds of values.

  • int → integer values (10, 20)
  • float → decimal values (10.5)
  • double → higher precision decimals
  • char → single character ('A')
  • bool → true or false
  • string → text values
int age = 20;
float marks = 88.5;
char grade = 'A';
string name = "Ravi";
bool isPass = true;
IO System

Input and Output

cin is used for input and cout is used for output.

int x;
cin >> x;
cout << x;

Multiple input example:

int a, b;
cin >> a >> b;
cout << a + b;
Logic Building

Operators in C++

  • Arithmetic → +, -, *, /, %
  • Relational → ==, !=, >, <, >=, <=
  • Logical → &&, ||, !
  • Assignment → =, +=, -=, *=
Revision

Key Takeaways

  • C++ is fast, powerful, and widely used in system-level programming
  • Every program starts from main()
  • Data types define memory and value type
  • cin and cout handle input/output
  • Operators are used for logic and calculations
