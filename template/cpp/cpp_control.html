C++ Control Flow & STL

Decision making, loops, arrays and standard template library

Control Statements

Decision Making (if-else)

Used to execute code based on conditions.

if (x > 10) {
    cout << "Large number";
} else {
    cout << "Small number";
}

Nested conditions improve decision complexity in programs.

Branching

Switch Case

Used when multiple fixed choices are available.

switch(n){
    case 1:
        cout << "One";
        break;

    case 2:
        cout << "Two";
        break;

    default:
        cout << "Other";
}

Break prevents fall-through execution.

Iteration

Loops in C++

Loops are used to execute code repeatedly.

For Loop

for(int i = 0; i < 5; i++){
    cout << i;
}

While Loop

while(x > 0){
    x--;
}

Do-While Loop

do {
    cout << x;
    x--;
} while(x > 0);

Loops are heavily used in competitive programming and interview coding questions.

Data Structure

Arrays

Arrays store multiple values in contiguous memory locations.

int a[5] = {1, 2, 3, 4, 5};

cout << a[0];   // first element

Important concept: indexing starts from 0.

Advanced Concept

STL (Standard Template Library)

STL provides ready-made data structures and algorithms.

Important Containers

  • Vector → dynamic array
  • Set → unique sorted elements
  • Map → key-value pairs
  • Stack → LIFO structure
  • Queue → FIFO structure

Vector Example

#include 

vector v;

v.push_back(10);
v.push_back(20);
v.push_back(30);

Sorting Example

#include 

sort(v.begin(), v.end());

Key Interview Point

STL reduces coding complexity and is widely used in technical interviews, especially in DSA-based rounds.

Key Takeaways

  • if-else is used for conditions
  • switch is used for multiple choices
  • loops are used for repetition
  • arrays store fixed-size data
