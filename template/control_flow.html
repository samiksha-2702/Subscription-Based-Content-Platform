Python Control Flow Cheat Sheet

Quick reference for loops and conditions

Conditional Statements

  • Simple if: if x > 0: print("Positive")
  • If-else: if x>0: ... else: ...
  • If-elif-else:
    • 
if x>0:
    print("Positive")
elif x<0:
    print("Negative")
else:
    print("Zero")

Loops

  • For loop: for i in range(5): print(i)
  • While loop: while x < 10: x +=1
    • 
for i in range(3):
    print(i)
else:
    print("Done")

Break & Continue


for i in range(5):
    if i==3:
        break
    print(i)

Match Case (Python 3.10+)


match day:
    case "Mon":
        print("Monday")
    case "Tue":
        print("Tuesday")
    case _:
        print("Other day")
