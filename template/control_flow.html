Conditional Statements

Simple if Statement

  • Used to execute code when condition is True
  • Syntax:

x = 10

if x > 0:
    print("Positive")

if - else Statement

  • Executes one block if condition is True
  • Else block runs when condition is False

age = 18

if age >= 18:
    print("Eligible")
else:
    print("Not Eligible")

if - elif - else

  • Used for multiple conditions
  • elif means "else if"

x = 0

if x > 0:
    print("Positive")
elif x < 0:
    print("Negative")
else:
    print("Zero")

Loops in Python

For Loop

  • Used to iterate through sequence
  • Works with range, list, tuple, string

for i in range(5):
    print(i)
  • range(5) gives values 0 to 4

While Loop

  • Runs until condition becomes False
  • Useful when iteration count is unknown

x = 1

while x <= 5:
    print(x)
    x += 1

Nested Loops

  • Loop inside another loop
  • Commonly used in patterns and matrices

for i in range(3):
    for j in range(2):
        print(i, j)

Loop Control Statements

Break Statement

  • Stops the loop immediately

for i in range(5):

    if i == 3:
        break

    print(i)

Continue Statement

  • Skips current iteration

for i in range(5):

    if i == 2:
        continue

    print(i)

Pass Statement

  • Placeholder statement
  • Used when code is required syntactically

for i in range(5):
    pass

Match Case (Python 3.10+)

Pattern Matching

  • Alternative to multiple if-elif statements
  • Introduced in Python 3.10

day = "Mon"

match day:

    case "Mon":
        print("Monday")

    case "Tue":
        print("Tuesday")

    case _:
        print("Other Day")
