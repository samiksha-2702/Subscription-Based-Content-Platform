Python Fundamentals

Variables & Data Types

  • Declare variable: x = 10
  • String variable: name = "PrepEdge"
  • Boolean value: is_active = True
  • Multiple assignment: a, b = 5, 10
  • Check type: type(x)
  • Delete variable: del x

Built-in Data Types

  • int → Integer values
  • float → Decimal values
  • str → Text/String
  • bool → True or False
  • list → Ordered mutable collection
  • tuple → Ordered immutable collection
  • dict → Key-value pairs
  • set → Unique unordered values

Input / Output

  • Print output: print("Hello")
  • User input: name = input()
  • Formatted output: print(f"Hi {name}")
  • Multiple values: print(a,b)
  • Separator: print(a,b,sep="-")

Operators & Expressions

Arithmetic Operators

  • Addition → +
  • Subtraction → -
  • Multiplication → *
  • Division → /
  • Modulus → %
  • Exponent → **
  • Floor Division → //

Comparison Operators

  • == Equal to
  • != Not equal to
  • > Greater than
  • < Less than
  • >= Greater or equal
  • <= Less or equal

Logical & Identity Operators

  • and
  • or
  • not
  • is
  • is not
  • in
  • not in

Type Conversion

Conversion Functions

  • Integer: int("10")
  • Float: float("10.5")
  • String: str(100)
  • Boolean: bool(1)
  • List: list((1,2,3))
  • Tuple: tuple([1,2])
  • Set: set([1,2,2])

Useful Built-in Functions

  • len() → Find length
  • max() → Largest value
  • min() → Smallest value
  • sum() → Sum of values
  • sorted() → Sort values
  • range() → Generate sequence

Comments in Python

  • Single line: # comment
  • Multi-line:
  • ''' comment '''
  • """ comment """
  • Comments improve code readability
