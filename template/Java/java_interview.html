Important Interview Differences

Concept Difference
JDK vs JRE JDK contains development tools while JRE only runs Java programs.
String vs StringBuilder String is immutable while StringBuilder is mutable and faster.
Array vs ArrayList Array has fixed size while ArrayList is dynamic.
Interface vs Abstract Class Interface supports full abstraction while abstract class allows partial abstraction.
Method Overloading vs Overriding Overloading changes parameters, overriding changes implementation.

Collections Framework

List

  • Ordered collection
  • Allows duplicates
  • Examples: ArrayList, LinkedList

Set

  • No duplicate values
  • Unordered collection
  • Examples: HashSet, TreeSet

Map

  • Stores key-value pairs
  • Keys are unique
  • Examples: HashMap, TreeMap
