Java Basics Cheat Sheet

Learn core Java fundamentals including JVM, variables, input handling, type casting, and syntax basics.

JDK vs JRE vs JVM

  • JVM (Java Virtual Machine) — Executes Java bytecode and makes Java platform independent.
  • JRE (Java Runtime Environment) — Contains JVM + libraries required to run Java applications.
  • JDK (Java Development Kit) — Includes JRE + compiler + debugging and development tools.
💡 JDK is used for developing Java applications, while JRE is mainly used for running them.

Hello World Program

Basic structure of a Java program: 


class Main {

    public static void main(String[] args){

        System.out.println("Hello Java");

    }

}
  • class Main → Defines the class name.
  • main() → Entry point of the program.
  • System.out.println() → Prints output to the console.

Variables & Data Types

Java is a statically typed language, so every variable must have a data type. 


int a = 10;
double price = 99.5;
char grade = 'A';
boolean flag = true;
String name = "Java";
  • int → Stores integers
  • double → Stores decimal numbers
  • char → Stores single characters
  • boolean → Stores true or false
  • String → Stores text

Operators


int a = 10;
int b = 5;

System.out.println(a + b); // 15
System.out.println(a - b); // 5
System.out.println(a * b); // 50
System.out.println(a / b); // 2
  • Arithmetic → + - * / %
  • Comparison → == != > < >= <=
  • Logical → && || !

Type Casting

Type casting converts one data type into another. 


int x = (int) 9.8;   // explicit casting
double y = x;        // implicit casting
  • Implicit Casting → Automatic conversion (small → large)
  • Explicit Casting → Manual conversion (large → small)

User Input using Scanner


import java.util.Scanner;

Scanner sc = new Scanner(System.in);

System.out.print("Enter number: ");

int n = sc.nextInt();

System.out.println(n);
  • Scanner class is used to take input from the user.
  • nextInt() → Reads integer input
  • nextLine() → Reads string input
