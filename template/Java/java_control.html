Java Control Statements

Learn Java decision making, loops, switch statements, and flow control concepts with examples.

If Else Statement

The if-else statement is used to execute different blocks of code based on conditions. 


int age = 18;

if(age >= 18){

    System.out.println("Adult");

}else{

    System.out.println("Minor");

}
  • if checks the condition.
  • else executes when the condition becomes false.
💡 Use if-else when your program needs decision making.

Switch Statement

The switch statement is used to select one block among multiple options. 


int day = 2;

switch(day){

    case 1:
        System.out.println("Mon");
        break;

    case 2:
        System.out.println("Tue");
        break;

    default:
        System.out.println("Invalid");

}
  • case defines different choices.
  • break stops execution after matching.
  • default runs if no case matches.

For Loop

The for loop is mainly used when the number of iterations is known. 


for(int i = 1; i <= 5; i++){

    System.out.println(i);

}
  • Initialization → int i = 1
  • Condition → i <= 5
  • Increment → i++

While Loop

The while loop executes as long as the condition is true. 


int i = 1;

while(i <= 5){

    System.out.println(i);

    i++;

}
  • Condition is checked before execution.
  • Useful when iteration count is unknown.

Do While Loop

The do-while loop executes at least once even if the condition is false. 


int i = 1;

do{

    System.out.println(i);

    i++;

}while(i <= 5);

Break & Continue

These statements control loop execution. 


for(int i = 1; i <= 10; i++){

    if(i == 5)
        break;

    if(i == 2)
        continue;

    System.out.println(i);

}
  • break exits the loop immediately.
  • continue skips the current iteration.

Nested Loop

A loop inside another loop is called a nested loop. 


for(int i = 1; i <= 3; i++){

    for(int j = 1; j <= 2; j++){

        System.out.println(i + " " + j);

    }

}

Important Interview Points

  • Difference between while and do-while loop
  • Use of break and continue
  • Nested loop logic
  • Switch statement flow
  • Infinite loops in Java
