What is SQL?

SQL (Structured Query Language) is used to store, manage and retrieve data from relational databases.

Database Terms

• Database — collection of tables
• Table — rows + columns
• Row — record
• Column — field

SQL Command Types

DDL → CREATE, DROP, ALTER
DML → INSERT, UPDATE, DELETE
DQL → SELECT
DCL → GRANT, REVOKE
TCL → COMMIT, ROLLBACK
