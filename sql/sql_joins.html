JOIN Meaning

Join combines rows from multiple tables using related columns.

INNER JOIN

Matching rows only SELECT * FROM A INNER JOIN B ON A.id=B.id;

LEFT JOIN

All left + matched right

RIGHT JOIN

All right + matched left

Constraints

PRIMARY KEY — unique + not null
FOREIGN KEY — reference key
UNIQUE — no duplicates
NOT NULL — must have value
