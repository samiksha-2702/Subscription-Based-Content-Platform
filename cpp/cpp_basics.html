What is C++?

C++ is a compiled, object-oriented programming language developed by Bjarne Stroustrup. It supports procedural + object-oriented + generic programming. Used in system software, games, compilers, embedded systems.

Basic Structure of C++ Program

#include <iostream>
using namespace std;

int main() {
    cout << "Hello World";
    return 0;
}
Explanation:
  • #include — library include
  • namespace std — avoid std:: prefix
  • main() — entry point
  • return 0 — success exit

Variables & Data Types

Common types:
  • int — integers
  • float — decimal
  • double — large decimal
  • char — single character
  • bool — true/false
  • string — text
int age = 20;
float marks = 88.5;
string name = "Ravi";

Input and Output

int x;
cin >> x;
cout << x;
Multiple input: 
cin >> a >> b;

Operators

  • Arithmetic → + − * / %
  • Relational → == != > <
  • Logical → && || !
  • Assignment → = += -=
