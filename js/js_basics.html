JavaScript Basics Cheat Sheet

Quick reference for JavaScript fundamentals

Variables

  • Declare variable: let x = 10
  • Constant: const pi = 3.14
  • Old style: var name = "Sam"
  • Block scope → let, const
  • Function scope → var

Data Types

  • Number → 10, 3.5
  • String → "Hello"
  • Boolean → true / false
  • Array → [1,2,3]
  • Object → {name:"A"}
  • Undefined → variable declared but not assigned
  • Null → intentional empty value
  • Check type → typeof x

Input / Output

  • Console output → console.log(x)
  • Alert popup → alert("Hello")
  • User input → prompt("Enter value")
  • Template string → `Hello ${name}`

Operators

  • Arithmetic → + - * / % **
  • Comparison → == === != !== > <
  • Logical → && || !
  • Assignment → = += -= *=
  • Type check strict → ===

Arrays

  • Create → let a = [1,2,3]
  • Access → a[0]
  • Add → a.push(4)
  • Remove → a.pop()
  • Length → a.length

Functions

  • Normal function:
    • function add(a,b){ return a+b }
  • Arrow function:
    • const add = (a,b) => a+b
  • Default parameter → function f(x=0)
