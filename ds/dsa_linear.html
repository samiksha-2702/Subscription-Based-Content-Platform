DSA — Linear Data Structures

Array • Linked List • Stack • Queue

Array

An array stores elements in contiguous memory locations and allows fast index-based access.

Operations Complexity

OperationTime
AccessO(1)
SearchO(n)
InsertO(n)
DeleteO(n)

Advantages

  • Fast access by index
  • Simple structure
  • Cache friendly

Disadvantages

  • Fixed size
  • Insertion costly
  • Memory wastage possible

Example

int arr[5] = {10,20,30,40,50}; print(arr[2]); // 30

Linked List

A collection of nodes where each node contains data and a pointer to the next node.

Types

  • Singly Linked List
  • Doubly Linked List
  • Circular Linked List

Operations Complexity

OperationTime
AccessO(n)
InsertO(1)
DeleteO(1)

Advantages

  • Dynamic size
  • Easy insertion/deletion
  • No contiguous memory needed

Example

Node → Data | Next Pointer 10 → 20 → 30 → NULL

Stack

Stack follows LIFO — Last In First Out.

Main Operations

  • push()
  • pop()
  • peek()

Complexity

OperationTime
PushO(1)
PopO(1)

Applications

  • Undo feature
  • Expression evaluation
  • Recursion call stack

Example

push(10) push(20) pop() → 20

Queue

Queue follows FIFO — First In First Out.

Main Operations

  • enqueue()
  • dequeue()
  • front()

Types

  • Simple Queue
  • Circular Queue
  • Priority Queue
  • Deque

Applications

  • CPU scheduling
  • Task processing
  • Printer queue

Example

enqueue(5) enqueue(8) dequeue() → 5
