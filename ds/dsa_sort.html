What is Sorting?

Sorting arranges data in a specific order (ascending or descending) to make searching and processing faster.

  • Improves search efficiency
  • Required in databases
  • Used in analytics & reporting

Bubble Sort

Repeatedly swaps adjacent elements if they are in wrong order.

Steps

  • Compare neighbors
  • Swap if needed
  • Repeat passes
for i in range(n): for j in range(n-i-1): if a[j] > a[j+1]: swap

Merge Sort

Divide and conquer algorithm.

  • Divide array into halves
  • Sort recursively
  • Merge sorted halves
mergeSort(arr): split merge(left,right)

Quick Sort

Uses pivot element and partitions array.

  • Select pivot
  • Partition
  • Recursive sort
pivot = arr[high] partition() quickSort(left) quickSort(right)

What is Searching?

Searching finds the location of an element in data.

  • Linear Search
  • Binary Search
  • Hash Search

Searching Complexity

MethodRequirementTime
Linear SearchUnsorted OKO(n)
Binary SearchSorted onlyO(log n)

Linear Search

Checks each element one by one.

for i in arr: if i == key: found

Use When

  • Small dataset
  • Unsorted data

Binary Search

Works only on sorted arrays.

  • Check middle
  • Discard half
  • Repeat
low=0 high=n-1 mid=(low+high)/2 if key < mid → left else → right

Advantages

  • Very fast
  • Logarithmic time

Real World Applications

  • Database indexing
  • Search engines
  • Leaderboards
  • Analytics sorting
  • Auto-complete systems
